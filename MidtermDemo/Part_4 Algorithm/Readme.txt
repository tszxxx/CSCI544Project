This is the folder contains the code scripts for the CSCI 544 Research Project
1. movieClassification_nb_training.py - Train a Naive Bayes Model and output the model coefficients
2. movieClassification_nb_testing.py - Test the NB model with the testing dataset
3. movieClassification_perceptron_train.py - Train a Perceptron Model and output the model coefficients
4. movieClassification_perceptron_test.py - Test the Perceptron model with the testing dataset
